About AzureSpeed

AzureSpeed.com is a speed test tool that measures your network latency, download, and upload speeds across multiple global Azure regions. You can use it find the best Azure region to get start.

This site is independently maintained by Blair Chen and other GitHub volunteers, start from 2013. This site is not officially associated with Microsoft. All data presented is sourced from publicly available APIs provided by Microsoft Azure. For more details on how this site is built, visit the AzureSpeed GitHub repository .

Funding and ads:
AzureSpeed is maintained during spare time and is personally funded; ads are displayed to help cover the costs.
