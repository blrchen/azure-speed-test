About AzureSpeed

AzureSpeed.com is a speed test tool that measures your network latency, download, and upload speeds across multiple global Azure regions. Use it to find the best Azure region to get started.

This site is independently maintained by Blair Chen and other GitHub volunteers since 2013. This website is not officially associated with Microsoft. All data presented is sourced from publicly available APIs provided by Microsoft Azure. For more technical details , visit the AzureSpeed GitHub repository .

Funding and ads:
AzureSpeed is maintained during spare time and personally funded; ads are displayed to help cover operational costs.
