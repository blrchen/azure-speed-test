About AzureSpeed

AzureSpeed.com is a speed test tool that measures your network latency, download, and upload speeds across multiple global locations. It is independently maintained by Blair Chen and the Github community in spare time, this site is not officially associated or endorsed by Microsoft. The source code is accessible on GitHub . This project is built using various open-source technologies such as:

  • Angular
  • ASP.NET Core
  • Azure IP Lookup
  • Azure SDK for JavaScript
  • Bootstrap
  • Cloud Catalog
  • d3-shape
  • ngx-charts
  • RxJS
