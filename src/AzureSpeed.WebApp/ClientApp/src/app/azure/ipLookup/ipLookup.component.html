Azure IP Lookup

The Azure IP Lookup tool allows quick searches for service tag and region information using either an IP address or domain name.

Restful API

Access the Azure IP lookup Restful API at https://www.azurespeed.com/api/ipinfo, with these examples:

  • Domain name: https://www.azurespeed.com/api/ipinfo?ipAddressOrUrl=www.azurespeed.com
  • IPv4: https://www.azurespeed.com/api/ipinfo?ipAddressOrUrl=104.45.231.79
  • IPv6: https://www.azurespeed.com/api/ipinfo?ipAddressOrUrl=2603:1030:0800:0005:0000:0000:BFEE:A418
