Frequently asked questions

What is Azure China and who operates it?

Azure China is operated by 21Vianet (Shanghai Blue Cloud Technology Co., Ltd), a wholly owned subsidiary of VNET Group. It's a physically separated instance of Azure services operated within China to ensure compliance with Chinese regulations.

How do I get access to Azure China?

To use Azure China, you need to:

  1. Create an account through azure.cn
  2. Provide required business documentation
  3. Complete real-name verification process
  4. Sign agreements with 21Vianet
What services differ between Azure global and Azure China?

Azure China offers many core Azure services, but availability may differ from global regions. Always consult the latest service availability matrix and documentation before planning deployments.
