About AzureSpeed

A speed test tool that measures network latency, download, and upload speeds across global Azure regions. Use it to discover the closest Azure region for your workloads.

Project information

AzureSpeed is an open source side project built and maintained by Blair Chen. This is not an official Microsoft product; all data comes from publicly available Azure APIs.

The source code is available on GitHub. Contributions, bug reports, and feature requests are welcome.
