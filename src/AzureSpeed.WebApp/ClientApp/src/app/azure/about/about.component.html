About

www.AzureSpeed.com - Azure speed test tool. Test your network latency, download and upload speed to Azure datacenters around the world.

www.AzureSpeed.com is maintained by Blair Chen(@blrchen), open source in GitHub, built on top of following open source projects
  • Angular
  • Application Insights JavaScript SDK
  • ASP.NET Core
  • Azure SDK for JavaScript
  • Bootstrap
  • d3-shape
  • Font Awesome
  • ngx-charts
  • RxJS
