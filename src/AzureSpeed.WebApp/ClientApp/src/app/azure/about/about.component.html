About

www.AzureSpeed.com is a speed test tool that measures your network latency, download, and upload speeds across multiple global locations.

Blair Chen maintains www.AzureSpeed.com as an open-source project on GitHub. It utilizes various open-source technologies such as

  • Angular
  • Application Insights JavaScript SDK
  • ASP.NET Core
  • Azure IP Lookup
  • Azure SDK for JavaScript
  • Bootstrap
  • Cloud Catalog
  • d3-shape
  • Font Awesome
  • ngx-charts
  • RxJS
